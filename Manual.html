01홈

→ route: home · layout: split
에디토리얼 랜딩 · 지도 좌 / 리스트 우 50:50

서비스의 첫 인상이자 가장 균형 잡힌 기본 화면. 좌측에 한국 지도(SVG 일러스트)가 펼쳐지고 우측에는 현재 영역의 건축물 카드 리스트가 위에서 아래로 흐릅니다. 상단에는 큰 헤드라인 "오늘은 어느 동네를 마실 다녀올까요?"와 타입(한옥·근현대·미술관 등) / 지역 필터 바.

  • 지도 핀 클릭 → 우측 리스트의 해당 카드가 하이라이트, 핀 주변에 펄스 애니메이션
  • 리스트 카드 클릭 → 지도 핀이 활성화 (양방향 동기화)
  • 하단에 에디터 큐레이션 3개 컬렉션 카드 밴드
layout: split 9 BUILDINGS · 3 COLLECTIONS

02지도

→ route: home · layout: mapPrimary
탐색 우선 · 지도 풀블리드 + 플로팅 패널

같은 home 라우트지만 레이아웃이 mapPrimary로 강제되는 모드. 지도가 화면 전체를 채우고, 좌측에는 NOW EXPLORING 패널(작은 건물 리스트), 우측에는 현재 선택된 건축물의 큰 카드가 떠 있는 형태입니다.

  • "지도 위에서 오늘 갈 곳을 골라요" — 탐색을 가장 빠르게 끝내는 모드
  • 우측 큰 카드 클릭 → 해당 건축물 상세 화면으로 진입
  • 상단 nav는 variant="transparent"로 지도 위에 떠 있음
layout: mapPrimary FULL-BLEED MAP

03건축물

→ route: buildings (인덱스) / detail (개별)
건축물 인덱스 + ArchDaily 톤 Editor's Picks + masilground 톤 그리드

상단 nav의 "건축물"을 누르면 인덱스 페이지로 진입합니다. 단일 건축물 페이지가 아니라 큐레이터가 수시로 고른 픽 + 전체 526곳을 펼친 리스트가 함께 보입니다.

  • EDITOR'S PICKS · 상단 고정 3개. 큰 카드 1 + 중간 카드 2 (ArchDaily 톤)
  • 전체 그리드 · 3-col 에디토리얼 그리드. 이미지 + "이름 / 건축가" + 카테고리 (masilground 톤)
  • TYPE / REGION 필터 (한옥, 근현대, 미술관 ··· · 서울, 경기 ···)
  • 카드 클릭 → route: detail 로 이동. 상세 화면은 레이아웃 변형 3종 (hero · sidebar · longform)
  • 상세 화면에서 "건축물 목록" 링크로 인덱스 복귀
2 routes: buildings + detail layout: hero / sidebar / longform 3 picks + 526 places

04코스

→ route: course
큐레이터가 묶은 마실 코스 인덱스 + 도슨트 예약

한 건축물이 아니라 "코스"를 보여주는 화면. 거리·소요시간·난이도·도슨트 유무·가격이 명확히 표기된 4개의 큐레이션 코스가 정렬됩니다.

  • C-01 북촌: 김수근 검은 벽돌 ↔ 종묘 회랑, 4시간 (도슨트 ₩32,000)
  • C-02 콘크리트와 빛: 1박 2일 미술관 순례 (₩280,000)
  • C-03 부석사: 가을 한정 당일 왕복
  • C-04 DDP 야간: 셀프 무료 투어

코스 카드 클릭 → 코스 상세 → 예약 플로우(도슨트 선택 → 토스 결제 → 외부 공간 연결)로 이어집니다.

05컬렉션

→ route: collection
에디터/큐레이터가 묶은 컬렉션 인덱스

코스가 "실제로 걷는 동선"이라면, 컬렉션은 "읽는 큐레이션". 큐레이터의 노트와 큰 건물 슬라이드로 구성된 컬렉션 6편을 펼쳐봅니다.

  • No.01 벽돌의 시간 — 한국 모더니즘 건축 8선 (이주영 · 14분)
  • No.02 한옥, 다시 짓다 — 21세기 한옥 12선 (박정현)
  • No.03 콘크리트와 빛 — 미술관 산책 (최은영)
  • No.04 바다 곁의 집들, No.05 도시의 틈, No.06 밤의 건축

레이아웃 변형 3종: magazine(큰 표지 + 에디터 노트) · index(잡지 차례) · bento(비대칭 그리드).

+2마실 시작하기 (온보딩)

→ route: onboarding · 4 step
처음 오는 사용자 · 가입 + 취향 등록 + 첫 큐레이션 선물

단순 회원가입이 아니라 "잡지를 처음 펴는 의식". 4단계 진행:

  • STEP 1 · 가입 (이름·이메일·비번·약관) 또는 소셜로 건너뛰기
  • STEP 2 · "어느 동네부터 마실 다녀올까요?" — 7개 지역 카드에서 하나 선택
  • STEP 3 · "어떤 건축이 끌리세요?" — 한옥/근현대/미술관 등 칩 다중 선택
  • STEP 4 · "첫 큐레이션이 도착했어요" — 취향에 맞는 컬렉션 1개를 선물처럼 제시 → "마실 시작하기 →" 로 홈 진입

step 1 하단에 "이미 회원이신가요? 로그인 →" 링크 제공. 두 화면이 서로 갈아탈 수 있게 연결.

4 steps 취향 → 큐레이션 매칭 완료 시 자동 로그인
