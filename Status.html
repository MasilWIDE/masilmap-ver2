PROJECT STATUS · MASILMAP VER2 · 2026-05-29

masilmap ver2 구현 현황 분석

masilmap ver2는 CDN 기반 React 18 + Babel standalone 정적 프로토타입입니다. 백엔드·DB·API가 존재하지 않으며, 모든 데이터는 src/data.jsx의 in-memory JavaScript fixture입니다. 본 문서는 실제 코드에 기반해 현재 구현 상태를 정리한 것이며, 아래 "미구현" 표시 항목은 디자인·UI만 시연되거나 코드에 존재하지 않는 기능을 가리킵니다.
